# nocms

content management system that doesn't suck

see [DESIGN](./DESIGN.md) for high-level design.

## how

### install

```
git clone https://github.com/ahdinosaur/nocms
cd nocms
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
